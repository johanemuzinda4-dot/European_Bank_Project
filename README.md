## 🛠️ Tech Stack & Workflow
- **Data Pipeline (`app.py`):** Automated data parsing, string sanitation, and absolute path handling using Python & Pandas.
- **Statistical Modeling (`analyze.py`):** Macro cohort segmentation identifying age (46–60) and regional (Germany) risk centers.
- **BI Interface (`dashboard.py`):** Interactive multi-page layout utilizing a top grid system, dynamic deltas, custom corporate palettes, and automatic target export engines.
- **CI/CD Deployment:** Version-controlled with Git and deployed live via Streamlit Community Cloud.

## 📈 Strategic Takeaways
1. **Product Saturation Risk:** Customers holding 3 or 4 financial products exhibit a drastic churn profile anomaly.
2. **Activity Engagement Gap:** Inactive client profiles account for double the risk exposure compared to active tiers.
3. **Targeted Demographics:** Identified high-net-worth client drop-offs primarily centered in German segments within the 46–60 age group.]
